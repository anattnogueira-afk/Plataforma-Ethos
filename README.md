# Plataforma-Ethos
Visão do Produto: Para empresas de tecnologia que não tem equipe compliance cujos usuários têm dificuldade de prever situações éticas futuras ou inéditas que podem causas problemas sensíveis para usuários e para a imagem da empresa. A Plataforma Ethos é uma plataforma de consulta sobre Éticas para empresas de tecnologia que expõe futuros problemas éticos e/ou inéditos e propõe uma tomada de decisão em um menor tempo diferentemente das soluções genéricas sem validação social e de plataformas com comunicação muito técnica. O nosso produto é uma trilha que exclui más decisões e atribui notas de impacto para cada parâmetro definido
